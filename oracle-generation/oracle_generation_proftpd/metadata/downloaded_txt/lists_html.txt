Mailing lists There are a number of mailing lists for the ProFTPD Project covering the
entire range of activites. They are hosted by SourceForge . How to unsubscribe List Policy To limit the amount of spam received on the lists, posting is
      restricted to list members only. The list administrators reserve the right to remove members from the
      list(s) at any time. Announcements A very low traffic mailing list, used for major announcements related to
ProFTPD (new versions, bug fixes, security issues). This list is
highly recommended for all ProFTPD users. Subscribe via e-mail or
visit the web interface . Users This list is for end user questions and comments on ProFTPD. This
is a high-volume list. Subscribe via e-mail or
visit the web interface . Developers This list is intended for those who are interested in the development and
future functionality of ProFTPD. Subscribe via e-mail or
visit the web interface . Docs This list is intended for those who are interested in the development of
documentation, including HOWTOs, tutorials, examples, and translations. Subscribe via e-mail or
visit the web interface . Committers This list receives the automated e-mail messages from CVS and the bug
tracking system. These messages are probably only of use or interest to
subscribers of the -devel list (it is intended to cut down on the traffic
hitting -devel). Subscribe via e-mail or
visit the web interface .
